# harmony-beta

### Getting Started:

1. Prerequisites:

        Python 3.9.6

2. Installation:

        git clone https://github.com/toneauaudio/harmony-beta.git

   
3. Dependencies:

         pip install pyo
         pip install wxPython

         or

         make install
